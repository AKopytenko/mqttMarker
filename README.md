Приложение для управление гео-метками на карте с тестовым прогоном через MQTT-брокер.

#### DEMO  
http://wr-team.ru/demo/mqtt-marker/

---

#### Установка зависимостей
    $ npm i

#### Запуск сборки на dev-сервере
    $ npm run dev

#### Собрать production
    $ npm run build
